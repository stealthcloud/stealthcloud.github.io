Virtual Lab Build

13 January 2021 • 1 min read

Lab build out for other considering running a home lab setup.

Lab parts list

note: I decided to build 2 nodes for personal reasons, a single node (half all parts) will be perfectly fine to use.

  • 2x ATX case - https://www.newegg.com/black-corsair-icue-220t-rgb-airflow-atx-mid-tower/p/N82E16811139142?Item=N82E16811139142
  • 8x 64GB LR Ram - https://www.newegg.com/p/1X5-003Z-018J5?Item=9SIA7S6B0B3748
  • 2x 2Tb M.2 - https://www.newegg.com/crucial-2tb-p2/p/N82E16820156274?Item=N82E16820156274
  • 2x Motherboard/CPU combo - https://www.newegg.com/supermicro-mbd-h11ssl-i302p-ma015-o-single-amd-epyc-7000-series-processor/p/N82E16813183698?Item=N82E16813183698
  • 2x PSU - https://www.newegg.com/corsair-rmx-series-rm1000x-1000w/p/N82E16817139140?Item=N82E16817139140
  • 2x CPU Cooler - https://www.newegg.com/noctua-nh-u12s-tr4-sp3-premium-grade-120mm-cpu-cooler-for-amd-tr4-sp3/p/13C-0005-00141?Item=9SIAADY63N0333 ######not required:
  • 2x 2 NIC AOC - https://www.newegg.com/p/14U-0032-000D1?Item=9SIACTND7R2549 (needed if using VDS in VCenter)
  • 4x 2Tb NAS Drives - https://www.newegg.com/red-wd20efrx-2tb/p/N82E16822236343?Item=N82E16822236343 (I had spare drives similar to these and threw them in for extra space)
  • 2x 1Tb WD SSD - https://www.newegg.com/western-digital-blue-1tb/p/N82E16820250088?Item=N82E16820250088

final build per node

256 Gb Ram, 16 Cores (32 threads), 2Tb NVMe storage, 4 Tb HDD storage. Cost: ~$3,000

OS

Test run

  • ESXI 7.0 U1
  • VCenter 7.0 U1
  • NSX-T 3.1
  • VSAN compatible but did not use
